# Interactive Sales Dashboard Report

## Project Overview
This project analyzes sales data using Python and creates an interactive dashboard with static and interactive visualizations.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly

## Features
- Data Cleaning
- Sales Analysis
- Product-wise Revenue
- Region-wise Revenue
- Monthly Sales Trend
- Correlation Analysis
- Interactive Dashboard

## Visualizations
- Bar Chart
- Box Plot
- Violin Plot
- Correlation Heatmap
- Line Chart
- Interactive Plotly Dashboard

## Business Insights
- Identified highest revenue product.
- Determined highest-performing region.
- Analyzed monthly sales trend.
- Explored numerical correlations.
- Visualized sales distributions.

## Conclusion
The dashboard provides meaningful insights into sales performance and demonstrates practical data visualization techniques using Python.
